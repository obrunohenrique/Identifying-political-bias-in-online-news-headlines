# Identifying-political-bias-in-online-news-headlines
PolitiBias-BR: Automated Political Bias Classification for Brazilian News. > An end-to-end NLP pipeline that leverages LLM-as-a-judge (Llama 3.1 via Ollama) for high-fidelity labeling and fine-tunes BERTimbau to detect ideological nuances across major and independent Brazilian media outlets.
